# 招标信息平台产品改进需求文档

_文档编号：PRD-2026-002 | 需求方：张玲志 | 承接方：小椰 | 日期：2026-03-18_

---

## 一、现状问题

当前 demo 和 prototype 偏**信息展示**，功能单一，缺少运营能力和市场竞争力。

### 现有功能
- ✅ 招标信息列表/卡片双视图展示
- ✅ 基础搜索、筛选（类型、地区、时间）
- ✅ 详情页信息展示

### 缺失功能
- ❌ 用户体系（登录、收藏、历史）
- ❌ 智能推荐/订阅
- ❌ 运营内容（推荐、专题）
- ❌ 增值服务（会员、VIP）
- ❌ 数据分析工具
- ❌ 社区/互动功能

## 三、产品定位

### 3.1 核心定位
**商业化招标信息服务平台** —— 不只是信息展示，还要有运营能力、变现能力、竞争壁垒。

### 3.2 目标用户
- B端：投标企业、销售人员、项目经理
- C端：个人投标者、自由职业者

### 3.3 商业模式
- 基础功能：免费（信息展示、搜索）
- 增值服务：VIP 会员（详细查看、导出、订阅）
- 企业服务：API 接口、定制化数据

## 四、功能需求清单

### 4.1 用户体系（优先级：高）

| 功能 | 说明 | 竞争力 |
|------|------|--------|
| 注册/登录 | 手机号、微信一键登录 | 留存用户 |
| 个人中心 | 头像、会员状态、账户管理 | 基础 |
| 我的收藏 | 收藏感兴趣的招标信息 | 核心功能 |
| 浏览历史 | 查看最近浏览记录 | 提升体验 |
| 消息通知 | 新标讯推送、会员通知 | 召回用户 |
| 标讯订阅 | 关键词订阅，自动推送 | 核心功能 |

### 4.2 智能推荐（优先级：高）

| 功能 | 说明 | 竞争力 |
|------|------|--------|
| 关键词订阅 | 设置关注关键词，符合条件自动推送 | 差异化 |
| 猜你喜欢 | 基于浏览/收藏记录智能推荐 | 粘性 |
| 行业标签 | 按行业（建筑、医疗、教育）分类推送 | 精准 |
| 地区偏好 | 关注特定地区的招标信息 | 精准 |

### 4.3 运营内容（优先级：中）

| 功能 | 说明 | 竞争力 |
|------|------|--------|
| 热门推荐 | 今日热门、即将截止、优质标的 | 流量入口 |
| 专题精选 | 专栏、热点解读、行业报告 | 内容价值 |
| 中标分析 | 中标企业分析、中标率统计 | 专业壁垒 |
| 行业报告 | 月度/季度行业趋势分析 | 增值内容 |

### 4.4 增值服务（优先级：高）

| 功能 | 说明 | 商业模式 |
|------|------|----------|
| VIP 会员 | 查看详细信息、导出、无限收藏 | 核心变现 |
| 竞价排名 | 企业置顶、优先展示 | 广告收入 |
| 数据导出 | Excel/CSV 批量导出 | 付费功能 |
| API 接口 | 给企业客户提供数据接口 | B端变现 |

### 4.5 数据工具（优先级：中）

| 功能 | 说明 | 竞争力 |
|------|------|--------|
| 竞品分析 | 某企业中标记录、竞争对手分析 | 专业壁垒 |
| 中标率统计 | 某地区/行业中标率分析 | 决策支持 |
| 趋势分析 | 行业热度、预算趋势 | 增值服务 |
| 企业画像 | 业主企业分析、历史合作 | 业务支持 |

### 4.6 互动社区（优先级：低）

| 功能 | 说明 | 竞争力 |
|------|------|--------|
| 评论区 | 招标信息讨论、经验分享 | 社区氛围 |
| 问答互助 | 投标问题解答 | 用户价值 |
| 业主评价 | 对招标方评分、评价 | 信息透明 |

## 五、信息架构改进

### 5.1 导航结构

```
首页
├── 热门推荐
├── 最新标讯
├── 即将截止
├── 我的订阅（需登录）
└── 猜你喜欢（需登录）

信息分类
├── 招标公告
├── 中标结果
├── 采购意向
├── 变更公告
├── 拟在建项目
└── 更多分类...

VIP 专区（需登录）
├── 热门推荐
├── 中标分析
├── 行业报告
├── 数据导出
└── 企业服务

我的（需登录）
├── 个人资料
├── 我的收藏
├── 浏览历史
├── 消息通知
├── 标讯订阅
├── VIP 会员
└── 设置
```

### 5.2 新增筛选维度

| 维度 | 选项 |
|------|------|
| 信息类型 | 招标公告、中标结果、采购意向、变更公告、废标公告、拟在建 |
| 行业分类 | 建筑工程、医疗卫生、教育科技、交通运输、能源化工... |
| 地区 | 全国/省/市/区（联动） |
| 预算区间 | 10万以下、10-50万、50-100万、100-500万、500万以上 |
| 项目规模 | 小型、中型、大型 |
| 发布时间 | 今天、近3天、近7天、近30天、自定义 |
| 业主类型 | 政府部门、事业单位、国有企业、私营企业 |
| 投标状态 | 报名中、已截止、已开标 |

## 六、会员体系设计

### 6.1 会员等级

| 等级 | 价格 | 功能 |
|------|------|------|
| 普通用户 | 免费 | 查看基本信息、每日3条详细、基础筛选 |
| 月度VIP | ¥99/月 | 无限查看、导出、订阅、消息推送 |
| 年度VIP | ¥899/年 | 月度VIP全部 + 优先客服、数据报告 |
| 企业版 | 定制 | API接口、定制数据、专属服务 |

### 6.2 VIP 专属功能

- 查看招标详细信息（联系人、联系方式、报名要求）
- 批量导出 Excel/CSV
- 关键词无限订阅
- 优先推送新标讯
- 竞品分析工具
- 中标率统计报告
- 专属客服
