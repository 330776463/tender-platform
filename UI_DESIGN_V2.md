# 招标信息平台 - 高保真UI设计规范

_Version: 1.0 | 设计：小椰 | 审核：小腾_

---

## 1. 设计理念

### 1.1 核心目标
- **信息高效获取** - 用户能在3秒内找到目标招标信息
- **专业可信** - 界面风格稳重、专业，建立用户信任感
- **操作便捷** - 减少操作步骤，提升用户体验

### 1.2 设计风格
- 简约现代，信息优先
- 清晰的信息层级
- 一致的视觉语言

## 9. 响应式设计

### 9.1 断点

| 断点 | 宽度 | 布局 |
|------|------|------|
| xs | < 480px | 单列，底部导航 |
| sm | 480-767px | 单列，侧边栏折叠 |
| md | 768-1199px | 双列布局 |
| lg | 1200-1599px | 三列布局 |
| xl | ≥1600px | 三列 + 侧边栏 |

### 9.2 响应式规则

```
桌面端 (>1200px):
- 卡片网格: 3列
- 侧边栏: 固定
- 列表: 完整信息展示

平板端 (768-1200px):
```

## 10. 动画规范

### 10.1 过渡时长

| 场景 | 时长 | 缓动函数 |
|------|------|----------|
| 颜色变化 | 200ms | ease |
| 位置变化 | 200ms | ease-out |
| 展开/收起 | 250ms | ease-in-out |
| 弹框 | 300ms | cubic-bezier(0.16, 1, 0.3, 1) |
| 页面切换 | 200ms | ease |

### 10.2 常用动画

```css
/* 淡入 */
@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}

/* 上浮 */
@keyframes slideUp {
  from { 
    opacity: 0;
    transform: translateY(10px);
  }
  to { 
    opacity: 1;
    transform: translateY(0);
  }
}

/* 骨架屏 */
@keyframes skeleton {
  0% { background-position: -200% 0; }
  100% { background-position: 200% 0; }
}
```

## 11. 组件状态规范

### 11.1 通用状态

| 状态 | 说明 | 视觉表现 |
|------|------|----------|
| default | 默认状态 | 正常样式 |
| hover | 悬浮状态 | 轻微变色、上浮 |
| active | 激活状态 | 按下效果 |
| focus | 焦点状态 | 聚焦边框 |
| disabled | 禁用状态 | 降低透明度 |

### 11.2 无障碍

- 所有交互元素需支持键盘操作
- 颜色对比度 ≥ 4.5:1
- 焦点状态需明显可见
- 图片需有 alt 文字
