## 爬取策略

### 第一阶段（优先级 ⭐⭐⭐）
1. **中国招标投标公共服务平台** - 官方权威，全行业覆盖
2. **中国政府采购网** - 政府采购主渠道
3. **省级政府采购网** - 区域详细信息（浙/粤/苏/沪/京）
4. **千里马招标网** - 商业平台，数据量大

### 第二阶段（优先级 ⭐⭐）
1. **铁路/交通工程交易中心** - 行业垂直
2. **医疗/教育招标网** - 行业垂直
3. **金融采购网** - 行业垂直

### 第三阶段（优先级 ⭐）
1. 其他行业垂直网站
2. 地方小平台

## 技术要求

### 爬虫框架
- Python 3.10+
- Scrapy / Playwright
- 异步爬取：aiohttp

### 数据存储
- MySQL 数据库
- 支持增量更新
- 去重机制

### 反爬应对
- 代理池
- 请求间隔
- User-Agent 轮换

## 交付要求

1. **数据源调研报告** - 各网站结构分析
2. **爬虫脚本** - 至少覆盖第一阶段
3. **数据入库** - 原始数据存储
4. **定时任务** - 增量更新机制
